# JavaScript 数据结构与算法

代码均采用 ES6 编写，使用 webpack 和 babel 将 ES6 自动转换成 ES5。

> 重点要掌握数据结构与算法的思想，编程语言只是一种实现工具。

## 测试环境

### 安装依赖
```bash
npm install
```

### 启动服务
```bash
npm run start
```

开启**测试环境**的服务后，可在 `src/index.js` 查看测试的代码，查看具体值输出。
